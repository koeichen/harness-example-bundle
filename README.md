# harness-engineering

A curated Git submodule map of open-source agent harness engineering references.

Start with [`AGENTS.md`](./AGENTS.md). The full study material lives in [`docs/reference-map.md`](./docs/reference-map.md) and [`docs/study-guide.md`](./docs/study-guide.md).

This repository studies how real agent systems implement:

- state machines
- event logs
- gates
- routing
- checkpoints
- human approval
- sandboxing
- tracing
- workflow orchestration
